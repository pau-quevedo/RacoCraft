Dos scripts per canviar com es veu el Racó.
Us heu de descarregar les extensions Stylus (pel css) i tampermonkey (pel js).

Per instal·lar (és molt fàcil, feu el mateix pels dos scripts):

  - Se us redirigeix a una pestanya per instal·lar l'estil/script de l'extensió que sigui.
  - Cliqueu install i hauria de funcionar :P
